# Alt Drag Resize

A Windows utility for alternative drag and resize functionality.

## Features
- Custom window dragging behavior
- Resize window handles
- Hotkey support

## Usage

Run `main.py` to start the application:

```bash
python main.py
```

To add to Windows startup:

```bash
python add_to_startup.py
```

## Requirements
- Python 3.6+
- Windows OS

## Files
- `main.py` - Main application
- `add_to_startup.py` - Startup registration utility
- `run.bat` - Batch file launcher
